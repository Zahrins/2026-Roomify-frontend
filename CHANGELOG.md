# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog
and this project adheres to Semantic Versioning.

---

## [v1.0.0] - 2026-02-16

### Added
- Initial frontend release using Vite + React + Tailwind CSS
- Responsive layout for desktop and mobile
- Landing Page
- User Authentication (Login & Register)
- Admin Authentication (Login & Register)
- User Dashboard
- Admin Dashboard
- Room Booking feature (create booking)
- Booking History page (Admin)
- Filter bookings by:
  - Status
  - Date
  - Search by borrower name (Admin)
- Building & Room detail pages
- Status badge styling (Pending, Approved, Rejected)
- Environment configuration using `.env`

### Improved
- UI refinement on Dashboard (User & Admin)
- Improved Building page layout
- Enhanced booking detail display
- Added phone number detail in booking information

### Fixed
- Fixed booking status filter logic
- Fixed UI alignment issues on Landing Page
- Minor routing issues between pages
